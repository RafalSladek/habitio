## How to Create Affiliate Partner Links

### 1. **Amazon Associates** 🔗

#### Step 1: Apply for Amazon Associates
1. Go to **https://affiliate-program.amazon.com/**
2. Click "Join Now"
3. Fill out application:
   - Website URL: `https://habitio.rafal-sladek.com`
   - Content type: "Habit tracking, personal development recommendations"
   - Traffic sources: "Organic SEO, in-app promotions"
4. Amazon reviews (usually instant approval for established sites)
5. Once approved, you get an **Associate ID** (format: `habitio-20`)

#### Step 2: Generate Amazon Affiliate Links

**Method A: Direct URL Generation**
```
https://www.amazon.com/[Product-ASIN]/dp/[ASIN]?tag=[ASSOCIATE-ID]
```

**Example:**
- Product: Apple Watch Series 9
- ASIN: `B0CCY53FPH`
- Associate ID: `habitio-20`
- Affiliate Link: `https://www.amazon.com/dp/B0CCY53FPH?tag=habitio-20`

**Method B: Amazon Link Shortener Tool**
1. Login to Amazon Associates dashboard
2. Click "Tools" → "Product Links"
3. Paste product URL or search ASIN
4. Generate short affiliate link (automatically tagged with your ID)
5. Example output: `https://amzn.to/3Qx1234`

**Method C: SiteStripe Browser Extension** (easiest)
1. Install "Amazon Associates SiteStripe" extension
2. Visit any Amazon product page
3. Click extension → "Get Link" (auto-generates affiliate link)
4. Copy and paste into your content

#### Step 3: Add Links to Content
Example in `docs/habits.md`:
```markdown
### Fitness Wearables for "Morning Exercise"

**Apple Watch Series 9**
- Tracks workouts, heart rate, sleep
- [Buy on Amazon](https://www.amazon.com/dp/B0CCY53FPH?tag=habitio-20)
- Commission: ~10% (watches category)
```

**Important:** Disclose affiliation in footer or header of page:
> *"This page contains affiliate links. We earn a commission when you purchase through these links at no extra cost to you."*

**Method A: Via Impact Dashboard**

2. Search for "Babbel" campaigns

4. Select link type:

- With ID: `https://go.impact.com/campaign/babbel-affiliate?aic=habitio-2026`

1. Standard referral URL: `https://www.babbel.com/?ref=[PARTNER-ID]`

## Link Management Best Practices

### Disclosure Requirements
✅ **Required on every page with affiliate links:**
```markdown
**Affiliate Disclosure:**
This page contains affiliate links. We earn a small commission when you 
purchase through these links at no additional cost to you. This helps 
support habit.io development while providing you with verified recommendations.
```

### Tracking & Analytics
**Set up UTM parameters for link tracking:**

```
https://www.amazon.com/dp/B0CCY53FPH?tag=habitio-20&utm_source=habitio-app&utm_medium=in-app&utm_campaign=fitness-wearables
```

**UTM Parameters:**
- `utm_source` = where link is placed (habitio-app, docs, email, etc.)
- `utm_medium` = type (in-app, web, email, social)
- `utm_campaign` = habit category (fitness-wearables, learning, meditation)

### Link Shortener Strategy
For long links, use a shortener but **preserve affiliate ID**:
- ❌ Bad: `https://bit.ly/abc123` (loses tracking)
- ✅ Good: `https://habitio.rafal-sladek.com/go/babbel` (custom redirect with affiliate ID embedded in backend)

**Setup custom redirects in your domain:**
```
/go/babbel → https://go.impact.com/campaign/babbel-affiliate?aic=habitio-2026
/go/amazon-watch → https://www.amazon.com/dp/B0CCY53FPH?tag=habitio-20
/go/headspace → https://www.headspace.com/?affid=habitio-2026&refer=habitio
```

## Legal & Compliance Notes

✅ **Must do:**
1. Disclose all affiliate relationships (FTC requirement)
2. Use `noopener noreferrer` on external links for security
3. Keep commission percentages updated if they change
4. Archive link performance data quarterly

❌ **Never do:**
1. Hide affiliate links (cloaking)
2. Use misleading CTAs ("Click here" without disclosing affiliation)
3. Promote products you don't believe in
4. Let affiliate links break (test monthly)
